# OptoView — Tabela Polarizada
Ferramenta clínica de optometria comportamental para TV 3D passiva.

## Publicar no GitHub Pages
1. Crie repositório público no GitHub
2. Faça upload dos 6 arquivos (index.html, manifest.json, sw.js, icon-192.png, icon-512.png, README.md)
3. Settings → Pages → branch main → Save
4. Aguarde 2-3 min → acesse https://SEU-USUARIO.github.io/REPOSITORIO/

## Instalar como app no tablet
- Android/Chrome: menu (3 pontos) → "Adicionar à tela inicial"
- iPad/Safari: compartilhar → "Adicionar à Tela de Início"
